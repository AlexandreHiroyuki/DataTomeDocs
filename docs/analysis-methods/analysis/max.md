# ⏫ Max

**Returns** the **maximum** value in the **array**.

### 📝 Syntax

```cpp
TypeOfArray max()
```

### 🔮 Example

```cpp
DataTomeAnalysis<int, long int> intAverage(10);

for(int i = 1; i < 5; i++){
    intAverage.push(i);
}

intAverage.max(); // Will return 4
```

### ⏱ Complexity

**Linear (O(n))** in the **point count**.
