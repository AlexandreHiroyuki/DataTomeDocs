# ⏬ Min

**Returns** the **minimum** value in the **array**.

### 📝 Syntax

```cpp
TypeOfArray minimum()
```

### 🔮 Example

```cpp
DataTomeAnalysis<int, long int> intAverage(10);

for (int i = 1; i < 5; i++) {
    intAverage.push(i);
}

intAverage.minimum(); // Will return 1
```

### ⏱ Complexity

**Linear (O(n))** in the **point count**.
